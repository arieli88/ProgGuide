# ProgGuide

פלטפורמת לימוד אינטראקטיבית לקורס **תכנות מתקדם C#** — שיעורים, תרגול, מבחן מסכם ואתגר סופי.

## התקנה והרצה

```bash
npm install
npm run dev
```

פתח [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## מבנה

- `src/content/chapters/` — תוכן שיעורים (13 פרקים)
- `src/data/practice/` — שאלות תרגול
- `src/data/final-exam.json` — מבחן מסכם
- `src/app/` — דפי האתר (Next.js App Router)
